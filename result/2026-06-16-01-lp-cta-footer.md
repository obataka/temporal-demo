## 設計の意図・クリティカルポイント

**個人情報ゼロ方針**  
個人名・苗字・住所等は元の HTML にも存在しなかったが、社名を「SOP Intelligence Platform」→「SOP Platform Labs」へ統一してフッターを一本化。

**クリティカルポイント 3 点**

1. **サポートポリシーブロック（新規）** — `📨 サポート体制について` の `bg-slate-800/60` ブロックが追加要素。Slack 等テキスト限定・音声 NG を明文化し、問い合わせ後のミスマッチを防ぐ。
2. **「無償PoC」へのフレーミング統一** — 「ヒアリング」ではなく「PoC（4 週間・週 2 時間・コアチーム伴走）」という具体的な価値提示に切り替え。申し込み障壁を下げつつ期待値を正確に伝える。
3. **構文検証 ExitCode 0 確認済み** — `html.parser` で構文エラーなし。Tailwind CDN 依存のため CSS の動作確認はブラウザで行うこと。

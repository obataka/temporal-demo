# lp.html — 「真の価値」セクション追加・CTA拡充プラン

## Context

前回セッション（2026-06-16）でCTA/フッターの文言を「無償PoC（4週間・週2時間）」に更新済み。
今回は「本システムが単なるMarkdown化ツールではなく、Temporal＋AIエージェントによる自律ガバナンス基盤である」
という核心を LP 内で力強く伝えるため、2点の修正を行う。

---

## 修正対象ファイル

`web-ui/public/lp.html` 1ファイルのみ

---

## 変更 1: 「真の価値」セクションの新規挿入

### 挿入位置

`<!-- ===== TRUST ===== -->` セクションの **直後**、`<!-- ===== CTA ===== -->` の直前

### セクション構成

- 背景: `bg-gray-50`（SOLUTIONセクションと同系色で連続感）
- ヘッダー: 「単なる手順書の Markdown 化ツールではない。」
- サブヘッド: 3つのステップが完全連動して初めて成立する基盤であることを表明
- 3カード（縦並び、左にアイコン・番号、右に説明）:

| # | タイトル | 説明 |
|---|---|---|
| 1 | 自律監査ループ | Writer/Reviewer AIエージェントが約7分間、セキュリティ・例外処理の観点から手順書のバグを炙り出す厳格な議論を自律実行 |
| 2 | 人間介入（HITL） | 監査中のエージェントに対し、人間がUIからいつでもリアルタイムに追記・修正指示を投入でき、エージェントが即座に再バリデーション |
| 3 | インフラ連動（自動PR） | 人間が承認した瞬間、Temporalと連動して修正済みの本物のコード（GitHub Pull Request）を自動生成し、インフラ更新を自動化 |

- カードデザイン: 白背景・左ボーダー色分け（emerald/amber/indigo）、番号バッジ
- セクション末尾に「3ステップが連動して初めて成立する唯一無二のシステム」という締め

### HTML概要

```
<section id="true-value" class="bg-gray-50 py-24">
  <div class="max-w-5xl mx-auto px-6">
    <!-- ヘッダー -->
    <div class="text-center mb-16">
      <p class="text-emerald-600 uppercase tracking-widest">Why Temporal × Multi-Agent AI</p>
      <h2>単なる手順書の Markdown 化ツールではない。</h2>
      <p class="text-slate-500">3つのステップが完全連動して初めて成立する基盤</p>
    </div>
    <!-- 3カード: 縦並び -->
    <div class="space-y-6">
      <!-- カード1: 自律監査ループ（emerald） -->
      <!-- カード2: HITL（amber） -->
      <!-- カード3: インフラ連動（indigo） -->
    </div>
  </div>
</section>
```

## 変更 2: CTAセクションの「無償 PoC に含まれるもの」拡充

### 追加項目（既存リストに2項目追加）

1. **Docker/プライベート環境デプロイ**  
   「本システムはお客様のプライベート環境（Dockerベース）へデプロイして検証を実施します。機密データが外部サーバーへ流出するリスクはゼロです。」

2. **既存手順書のコンバート代行**  
   「現在運用中の手順書（Excel / Word / PDF 等）をメッセージで送るだけで、検証用データへのコンバートからエージェント監査のキックまで、当方が完全リモートで代行します。」

### 挿入位置

既存 `週 2 時間を上限とした` の項目の**後**（リストの最後に2項目追加）

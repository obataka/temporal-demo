**画面：** エージェントステータスバッジエリアにフォーカス。

```

[Writer] 代替構成パターンを3案生成中...
```

- ログ末尾に点滅カーソル `_` が明滅

**テロップ（右下 or 上部）：**

```

```

**ナレーション：**  
> 「Writer エージェントが起動しました。差し戻しコメントを解釈し、ドラフトの再構成を自律的に開始しています。」

**編集指示：** ログ流れの速さは **実収録のまま（等倍速）** で使用。視聴者にリアリティを感じさせる。

### 1:35–1:55 ── Writer → Reviewer バトン（アンバーパルス明滅）

**画面：** バッジエリアを横断するトランジションを強調。

```

[Reviewer] 緊急連絡先フローの具体性 → 合格
[Reviewer] Section 3 の冗長表現を検出。改善提案を Writer に送信...
```

**テロップ：**

```

```

**ナレーション：**  
> 「今度は Reviewer エージェントが、Writer の出力を批評します。承認か、差し戻しか——AIが自律で判断します。」
